> npx -y @guildai/cli@0.12.3 agent init --name dailygate-agent --template LLM --directory .

## What lives here after init
- `agent.ts` — the `llmAgent({ description, tools, systemPrompt, mode })` definition.
- `guild.json` — Guild project manifest.
- `package.json` — pulls the Guild SDK (not a public npm package; init wires it).

## What WE add on top of the scaffold
```
tools/
  context.ts      → calls Person B's GET /context  (GATHER + read trust)
  decision.ts     → calls Person B's POST /decision (record + upsert)
  composio.ts     → Composio actions (assign / close-dup / email / Slack post)
policy/
  classify.ts     → category + stakes + reversible + confidence for an event
  decide.ts       → act-vs-escalate branch (reads trust_level, hard ceiling)
trace/
  langfuse.ts     → spans around classify → branch → trust-update
```

## The act-vs-escalate rule (the spine)
```
if (stakes HIGH && irreversible)            → ALWAYS escalate   (hard ceiling)
else if (trust[category]=="auto" && confident) → ACT (Composio)
else                                        → escalate (ui_prompt)
```
